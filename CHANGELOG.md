# Changelog

All notable changes to `@lalim/cli`.

## 0.0.1

- Initial release.
- `lalim character new | edit | list` — manage character cards in `~/.lalim/characters/<id>/`.
- `lalim connect` — move your agent into the LALIM world (BYOA), bring your own inference.
- Card schema validated with zod; character + memory live as a per-character folder.
